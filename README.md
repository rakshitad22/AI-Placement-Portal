# AI Placement Portal

An AI-powered Placement Preparation and Resume Analysis Portal designed to help students prepare for placements efficiently.

## Features

* User Registration and Login
* AI Resume Analyzer
* ATS Score Prediction
* Job Recommendations
* Aptitude Tests
* DSA Practice Questions
* Progress Tracking Dashboard
* Leaderboard System
* AI Interview Assistant
